## Requirements
Here is the list of libraries you need to install to execute the code:
* Python 3.6
* PyTorch 1.7
* scikit-image
* scipy
* cv2 (opencv for python)

All of them can be installed via `conda` (`anaconda`), e.g.
```
conda install scikit-image
```

## How to Execute Demo
Run the demo code in `train.py`.
